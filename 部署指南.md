# 金星食谱管理系统 - GitHub 部署指南

## 🚀 快速部署步骤

### 1. 打开 GitHub 部署文件夹
```
/Users/spikesu/Desktop/金星食谱/venus-recipe-github/
```

### 2. 上传到 GitHub
1. 在 GitHub 创建新仓库
2. 将 `venus-recipe-github` 文件夹内容上传到仓库

### 3. 启用 GitHub Pages
1. 进入仓库设置 (Settings)
2. 找到 Pages 选项
3. Source 选择 "Deploy from a branch"
4. Branch 选择 "main"
5. Folder 选择 "/ (root)"
6. 点击 Save

### 4. 访问网站
部署完成后访问：`https://你的用户名.github.io/仓库名`

## ✅ 功能特性

### 核心功能
- ✅ **5餐制管理**: 早餐、上午加餐、午餐、下午加餐、午点
- ✅ **多园区支持**: 同时管理多个园区
- ✅ **营养统计**: 杂粮、水果、肉类使用统计
- ✅ **食谱表**: 周视图食谱展示
- ✅ **响应式设计**: 支持手机和电脑

## 🆘 常见问题

### Q: 中文字符显示异常？
A: 确保文件保存为 UTF-8 编码格式

### Q: 页面无法加载？
A: 检查文件路径是否正确，确保所有文件都已上传

### Q: 功能无法使用？
A: 这是演示版本，部分功能为展示用途

## 📞 支持

如有问题，请检查：

2. 文件是否完整上传
